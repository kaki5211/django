    カテゴリー（青春/白秋）

    {% comment %}
    {% endcomment %}

    　このブログでは、カテゴリーとは別に、Young/Oldで、大きく２つに分類してる。

    　これは、主人公と登場人物が、若い/若く無いを、意味してる。

    　この方が、後々紹介する本が多くなってきたときに、目的の本に辿りやすいかなと思ったから、独自に追加した。

    　とりあえずチャレンジ。

    　以下の表が、ざっくりした定義。

    カテゴリー（青春/白秋） カテゴリー説明
    青春 主人公や周りの登場人物の年齢が、大体０～２９歳で構成される場合、
    青春/白秋カテゴリーが「Y」（Young）に分類される
    白秋 主人公や周りの登場人物の年齢が、大体２９歳以上で構成される場合、
    青春/白秋カテゴリーが「O」（Old）に分類される
